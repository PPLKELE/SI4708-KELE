2. STRUKTUR PROYEK
------------------------------------------------------
Aplikasi dibangun menggunakan Arsitektur Fullstack:
- Folder "frontend/" : Berisi kode antarmuka / UI berbasis React.js (Vite).
- Folder "backend/"  : Berisi server Node.js dan Express API.
- Database MySQL : Aplikasi menggunakan MySQL server (perlu XAMPP/MySQL yg menyala). Database "work4village_db" dan isinya akan dibuat secara otomatis.


3. CARA MENJALANKAN (RUNNING) APLIKASI DI KOMPUTER LAIN
------------------------------------------------------
Pastikan Anda sudah menginstal "Node.js" dari situs resminya (nodejs.org) sebelum menjalankan aplikasi.
Pastikan juga service MySQL (misal melalui aplikasi XAMPP) sudah di-Start dan berjalan.

Langkah A: Menjalankan Backend (Server API)
1. Buka Terminal / CMD / PowerShell.
2. Masuk ke direktori backend:
   cd "backend"
3. (Opsional bila belum pernah instal dependency) Jalankan:
   npm install
4. Jalankan server backend:
   node server.js
5. Pastikan muncul pesan: 
   "Database work4village_db checked/created." dan "Connected to MySQL work4village_db."
   serta "Work4Village Server is running on port ... "

Langkah B: Menjalankan Frontend (Tampilan Web UI)
1. Buka jendela Terminal / CMD baru (biarkan terminal Backend tetap berjalan).
2. Masuk ke direktori frontend:
   cd "frontend"
3. (Opsional bila belum pernah instal dependency) Jalankan:
   npm install
4. Jalankan development server frontend:
   npm run dev
5. Terminal akan menampilkan URL: http://localhost:5173
6. Buka Google Chrome atau browser lain lalu kunjungi URL tersebut.
